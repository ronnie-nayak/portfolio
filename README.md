## Run Locally

Clone the project

```bash
  git clone https://github.com/ronnie-nayak/portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Build the Project

```bash
  npm run build
```

Run Build Files

```bash
  npm run start
```
